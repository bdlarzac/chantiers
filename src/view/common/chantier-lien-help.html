Unité de gestion
Vous devez saisir un code d'UG (ex: XIX-11).
Si vous souhaitez associer le chantier avec plusieurs UGs, cliquez sur et saisissez un autre code UG.
Lorsque vous avez choisi les UGs, cliquez sur , pour faire apparaître les fermiers et les lieux-dits correspondant à ces UGs.
Vous pouvez à tout moment supprimer une UG (avec ) ou en ajouter d'autres (avec ), mais il faut ensuite valider à nouveau (avec ) pour que la liste des fermiers et lieux-dits se mette à jour.
