Pourcentage SCTL / GFA dans les ventes de plaquettes

La structure des données est la suivante :
  • Une vente de plaquettes donne lieu à une ou plusieurs livraisons.
  • Chaque livraison donne lieu à un ou plusieurs chargements.
  • Un chargement se fait sur un seul tas.
  • Un tas est relié à un seul chantier plaquettes.
  • Un chantier est relié à des parcelles.
La surface de chaque parcelle concernée par le chantier est saisie dans le logiciel.

On utilise la surface des parcelles pour calculer la proportion SCTL / GFA dans chaque tas.

Exemple

Chantiers

On a 2 chantiers plaquettes qui ont été saisis de cette manière :

Chantier 1
Associé à 3 parcelles :
  • Parcelle 1 - surface totale = 10 ha ; propriétaire = GFA ; le chantier concerne 2 ha
  • Parcelle 2 - surface totale = 3 ha ; propriétaire = SCTL ; le chantier concerne toute la parcelle
  • Parcelle 3 - surface totale = 15 ha ; propriétaire = SCTL ; le chantier concerne 5 ha
Ce chantier a permis de broyer 1000 maps, qui ont été stockées dans 2 tas :
  • 200 maps dans le Tas 1 au hangar des Baumes
  • 800 maps dans le Tas 2 au hangar des Liquisses
Les parcelles permettent de calculer la proportion SCTL / GFA de ce chantier (et donc des tas):
Surface totale = 2 + 3 + 5 = 10 ha
Surface SCTL = 3 + 5 = 8 ha => pourcentage = 8 / 10 = 80 %
Surface GFA = 2 ha => pourcentage = 2 / 10 = 20 %

Chantier 2
Associé à 2 parcelles :
  • Parcelle 4 - surface totale = 8 ha ; propriétaire = SCTL ; le chantier concerne 2 ha
  • Parcelle 5 - surface totale = 4 ha ; propriétaire = SCTL ; le chantier concerne toute la parcelle
De la même manière, on calcule les pourcentages SCTL / GFA :
SCTL = 100 %
GFA = 0 %

Ce chantier a permis de broyer 500 maps, qui ont été stockées dans 2 tas :
  • 100 maps dans le Tas 3 au hangar des Baumes
  • 400 maps dans le Tas 4 au hangar des Liquisses
Pour récapituler, on a :
% SCTL% GFA
Tas 1 200 maps 80 % 20 %
Tas 2 800 maps 80 % 20 %
Tas 3 100 maps 100 % 0 %
Tas 4 400 maps 100 % 0 %

Vente

On fait une vente de 100 maps, composée de 2 livraisons de 50 maps.

Livraison 1, pour laquelle on fait 2 chargements au hangar des Baumes :
  • 20 maps sur le tas 1
  • 30 maps sur le tas 3
Livraison 2, pour laquelle on fait 2 chargements au hangar des Liquisses :
  • 10 maps sur le tas 2
  • 40 maps sur le tas 4
On peut maintenant calculer les parts SCTL / GFA dans la vente.
Vente Chantiers
% dans les tas 		Résultat
% dans la vente
SCTL GFA SCTL GFA
Tas 1 20 maps 20 % 80 % 20 % 16 % 4 %
Tas 2 10 maps 10 % 80 % 20 % 8 % 2 %
Tas 3 30 maps 30 % 100 % 0 % 30 % 0 %
Tas 4 40 maps 40 % 100 % 0 % 40 % 0 %
Total 100 maps 100 % 94 % 6 %
Par ex, pour le tas 1 :
% SCTL dans la vente = 20 % x 80 % = 20/100 x 80/100 = 0.2 * 0.8 = 0.16 = 16 %
% GFA dans la vente = 20 % x 20 % = 20/100 x 20/100 = 0.2 * 0.2 = 0.04 = 4 %
Si la vente fait 1000 €, on comptera donc 940 € pour la SCTL et 60 € pour le GFA.
